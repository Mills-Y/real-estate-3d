# Model Info Panel - Quick Reference

## Panel Controls

| Button | Icon | Action |
|--------|------|--------|
| **Minimize** | ⊟ | Collapse panel to 50px tab |
| **Maximize** | ⊞ | Expand panel from collapsed state |
| **Close** | × | Hide panel completely |
| **Analytics** | 📊 | Toggle panel visibility (header) |

## Information Sections

### 📄 Basic Information (Default: Expanded)
- File format (GLTF, GLB, OBJ, etc.)
- File size (MB/KB)
- Upload date and time

### 📦 3D Model Properties (Default: Collapsed)
- **Dimensions**: Width × Height × Depth (meters)
- **Bounding Box**: Min/Max coordinates
- **Geometry**: Polygon count, Vertex count
- **Textures**: Texture count

### ⚡ Performance Metrics (Default: Collapsed)
- Load time
- Render time
- Memory usage

### 🎯 Quick Actions
- Export Model Data
- View Full Technical Details

### 📊 Current Session
- Views in this session
- Time viewing

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `ESC` | Close viewer and return to dashboard |

## Panel States

### 🔲 Expanded (350px, resizable)
Full information display with all sections.

### 📏 Collapsed (50px)
Vertical tab with "MODEL INFO" text and controls.

### ❌ Closed (Hidden)
Panel completely hidden, full screen for viewer.

## Display Modes

### Sidebar Mode (Default)
- Docked to right side
- Resizable (250-600px)
- Minimize/maximize support

### Floating Mode
- Compact floating widget
- Bottom-right corner
- Switch with Wind icon 🌬️

## Common Actions

### Resize Panel
1. Hover over left edge
2. Drag left/right (250-600px)

### Toggle Section
- Click section header to expand/collapse

### Switch Mode
- Click Wind icon (🌬️) for floating
- Click Layout icon for sidebar

## Tips

💡 **Maximize Space**: Use minimize instead of close for quick access  
💡 **Check Dimensions**: Review before exporting models  
💡 **Monitor Performance**: Watch polygon count for optimization  
💡 **Export Data**: Save technical specs for documentation
